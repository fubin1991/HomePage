大家好！我是一位在读于香港大学物理系的PhD学生，非常荣幸在香港大学得到了 王健教授 ( Prof. Jian Wang ) 的指导。

我的研究方向是量子输运领域，研究兴趣主要集中在提出计算各种输运物理量的理论方法和优化量子输运的各种算法。 随着互联网和信息技术的发展，我也开始关注机器学习技术在学术界的广泛应用。

香港大学，理学院，物理学系，凝聚态方向，香港，2014年9月 - 至今

研究课题

用快速多极子算法 ( Fast Multipole Method, FMM ) 提高时间相关的量子输运现象的计算性能， 主要用于计算无相互作用系统中的暂态电流。

用全系综统计 ( Full Counting Statistics, FCS ) ，计算无序系统的量子输运过程中相关的物理量。

课程

在这一学期(2016-2017   Spring)，我是王健教授 开设的计算物理 ( Computational Physics ) 课程的助教。 这门课程主要介绍在物理研究中常用的计算方法和计算理论，重点介绍了数值积分，数值微分，插值算法，微分方程和矩阵计算等内容。 如果想了解详细情况，请关注我们的 课程网页。

在2014至2016年，我曾是工科物理 ( Physics for Engineering Student ) 课程的助教。
